How to install Prism. (If you don't know how to.)




How to install Prism on Windows:



1. Click the Windows button and install the Setup file / .exe file.


2. Double click on the .exe File and follow the steps on the screen.


3. At the end, make sure the there's a check near the Option: "Launch Prism" and, click Finsh.




How to install Prism on MacOS:



1. Control+click on the app file and select "Open".


2. If a warning appears, select "Open" if it's an option.


3. If a warning appears but "Open" isn't an option, press "Cancel" and repeat from step 1.


The open button will appear the second time the warning appears.
